# 🐕 Shelter Manager – умный помощник для усыновления животных

## 📦 Стек технологий

- **Frontend:** React, TypeScript, TailwindCSS, React Router
- **State management:** Zustand / Context API
- **Backend:** Node.js + Express
- **Database:** PostgreSQL + Prisma ORM
- **Testing:** Jest, React Testing Library (frontend), Supertest (backend)
- **Notifications:** Nodemailer (email), in-app уведомления
- **Other:** Vite, ESLint, Prettier

## 📋 Описание проекта

**Идея:**  
«Shelter Manager» — это веб-платформа, которая помогает приютам эффективнее пристраивать животных. Главная инновация — **умный помощник для усыновителей**. Человек заполняет короткую анкету (есть ли дети, другие питомцы, тип и площадь жилья, опыт содержания), а система рекомендует питомцев с наибольшей вероятностью прижиться в этой семье. Это снижает возвраты животных и экономит часы работы сотрудников.

**Дополнительные возможности:**
- Полный жизненный цикл животного (поступление → лечение → готов к пристройству → пристроен)
- In-app и email‑уведомления (заявка одобрена, новая задача волонтёру)
- Волонтёрские задачи с напоминаниями
- Карточки животных с фото, характеристиками и историей прививок (опционально)
