# Проблема и решение

**Проблема:**  
Приюты тратят огромное время на подбор животных для усыновителей. Сотрудники вынуждены вручную опрашивать кандидатов и перебирать карточки питомцев. Из-за этого:
- Хорошие животные месяцами ждут семьи
- Часты возвраты (животное не подошло по характеру или условиям)
- Усыновители испытывают стресс от выбора

**Решение:**  
Создать интеллектуального помощника, который:
1. Собирает от усыновителя анкету (дети, другие животные, жильё, опыт).
2. Сопоставляет с характеристиками каждого животного (дружелюбность, потребность в пространстве, активность).
3. Выдаёт список питомцев с **процентом совместимости** и пояснениями.

**Дополнительная ценность:**  
- Единая база животных с их статусом (лечится, на карантине, готов к усыновлению)
- Автоматические уведомления (не пропустить важное)
- Волонтёрские задачи в одном окне
- Прозрачность для усыновителей (видят историю и прививки)

**Почему это важно для НКО / бизнеса:**  
Ускорение пристройства → экономия ресурсов → больше спасённых жизней.
