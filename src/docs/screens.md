# Ключевые экраны приложения

## 1. Главный экран (для гостя / усыновителя)
- Крупная кнопка **«Найти своего питомца»** (ведёт к анкете)
- Список животных со статусом `READY_FOR_ADOPTION` (случайные или популярные)
- Фильтры по виду (кошки/собаки)

## 2. Анкета усыновителя
- Форма с шагами:
  - Есть ли дети?
  - Есть ли другие животные? (если да — какие)
  - Тип жилья (квартира/дом) + площадь (малая/средняя/большая)
  - Ваш опыт содержания (нет / начальный / опытный)
- Кнопка «Показать рекомендации»

## 3. Страница рекомендаций
- Список карточек животных, отсортированный по проценту совместимости (убывание)
- На каждой карточке: фото, имя, крупный процент (например, 94%), статус (только READY_FOR_ADOPTION)
- Кнопка «Подробнее» → переход в карточку животного

## 4. Карточка животного (для усыновителя)
- Фото, основные характеристики (возраст, вид)
- **Блок «Совместимость с вами»** (если заполнена анкета): процент и пояснение («Отлично подходит для квартиры с детьми»)
- Блок «Забронировать» → подача заявки
- Дополнительно (на вкладке): история прививок, здоровье

## 5. Личный кабинет сотрудника / администратора
- Список животных (все статусы) с возможностью редактировать
- Кнопка «Добавить животное» (форма с полями характеристик)
- Список заявок с кнопками «Одобрить» / «Отклонить»
- Список волонтёрских задач, календарь
- Статистика по рекомендациям (сколько животных пристроено через алгоритм)

## 6. Центр уведомлений (иконка колокольчика)
- Выпадающий список последних уведомлений (непрочитанные выделены)
- Страница «Все уведомления» с фильтром по типу

## 7. Страница волонтёра (отдельный дашборд)
- Список открытых задач на сегодня/завтра
- Мои взятые задачи (in progress) с кнопкой «Выполнить»

## Адаптивность:
- На телефоне: главное меню — гамбургер, анкета в один столбец, карточки — вертикальный список.
- Все модальные окна удобны для тапа.
